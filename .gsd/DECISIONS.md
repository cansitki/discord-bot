| D009 | M001/S05 | pattern | How cogs expose tools to Claude via ai.py | Duck-typed protocol: any cog with `get_tools() -> list[dict]` and `handle_tool_call(name, input, message) -> str` methods gets its tools automatically included in Claude API calls | The hardcoded `ServerDesignCog` check in ai.py doesn't scale to multiple tool-providing cogs.
